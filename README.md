# 🎉 Birthday Surprise App

A small interactive birthday webpage built with **Next.js**, **TypeScript**, and **Framer Motion**, created as a fun surprise project for a friend’s birthday.

It features animations, confetti effects, and a personalized birthday message to make the celebration feel a bit more magical ✨

---

## 🎯 Purpose

This project was created as a small personal gift — a fun way to celebrate a friend’s birthday using code instead of a traditional card.

---

## ✨ Preview

When you open the app, you’ll see:
- A full-screen animated gradient background
- Falling confetti 🎊
- A floating birthday card with smooth animations
- A profile image with subtle motion effects
- A personalized birthday message 

---

## 🚀 Tech Stack

- [Next.js](https://nextjs.org/) – React framework
- [TypeScript](https://www.typescriptlang.org/) – Type safety
- [Framer Motion](https://www.framer.com/motion/) – Animations
- [react-confetti](https://www.npmjs.com/package/react-confetti) – Confetti effect
- Tailwind CSS – Styling
